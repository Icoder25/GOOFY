# Goofy — AI OS Agent

Goofy is a system-wide, voice-controlled AI Operating System Agent. It lives in your system tray, listens natively with zero latency, and uses Playwright and OS-level hooks to physically execute tasks on your computer.

It is capable of instantly interacting with any active browser window, taking screenshots, scrolling, and parsing complex intents via a local Gemini-powered FastAPI backend.

### Quick Start
If everything is already installed:
1. Double click `desktop/run_goofy.bat`
2. Say: *"Hello Goofy, take a screenshot"*
